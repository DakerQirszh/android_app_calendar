# 智程日历（Smart Schedule Calendar）

智程日历是一款基于 **Android 原生技术** 开发的个人日程管理应用，面向学习、工作与日常生活场景，提供多视图日历展示、日程管理以及系统级提醒功能。项目以课程设计为背景，重点关注功能完整性、工程结构清晰性与技术实现规范性。

---

## 功能特性

### 基本功能
- **多视图日历展示**
  - 月视图：整体浏览月份日期分布
  - 周视图：按周查看日期并支持前后切换
  - 日视图：集中管理某一天的日程列表
- **日程管理**
  - 日程的添加、查看、编辑与删除
  - 支持日程分类（工作 / 学习 / 生活 / 提醒等）
  - 支持完成状态标记
- **日程提醒**
  - 基于系统服务的定时提醒
  - 到达提醒时间后通过通知提示用户
  - 删除或完成日程后自动取消对应提醒

### 扩展功能
- **农历日期展示**
  - 本地算法计算农历日期
- **星座信息显示**
  - 根据所选日期展示对应星座
- **日程搜索**
  - 支持按标题或描述进行关键词搜索

## 技术栈

- **开发语言**：Kotlin  
- **UI 框架**：Jetpack Compose  
- **本地存储**：Room（SQLite）  
- **异步处理**：Kotlin Coroutines  
- **系统服务**
  - AlarmManager
  - BroadcastReceiver
  - NotificationManager  

---

## 项目架构

项目整体采用分层架构设计，主要包括：

- **UI 展示层**
  - 基于 Jetpack Compose 的声明式界面
  - 月 / 周 / 日视图与对话框组件
- **业务与数据层**
  - Room 数据库
  - Event 实体与 EventDao 数据访问接口
- **提醒与系统服务层**
  - ReminderScheduler
  - AlarmReceiver
  - NotificationHelper
- **辅助模块**
  - 农历计算
  - 星座计算
  - 日程分类颜色映射

该架构实现了职责清晰、依赖单向的模块组织方式，具备良好的可维护性与扩展性。
